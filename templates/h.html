REST API Framework for Test

When you go to the home page ("/"), you will get information about REST API calling.

Endpoints:

/persons: Returns all information in a table with tabular output in the CLI.

/api/books/person: Returns user information in REST API format.

/api/books/person/<int:id>: Returns specific user information based on the provided ID.

Posting User Information:

/api/books/person (POST): Allows posting user information in JSON format.

        {
            "name": "monika",
            "age": 23,
            "gender": "female"
        }

Updating User Information:

/api/books/person/<int:id> (PUT): Updates information for a specific user identified by the provided ID. Example curl command:

        curl -X PUT -H "Content-Type: application/json" -d "{\"id\": 1, \"title\": \"Updated Title\", \"author\": \"Updated Author\"}" http://127.0.0.1:5000/api/books/persons

Deleting Specific User Information:

/api/books/person/<int:id> (DELETE): Deletes information for a specific user identified by the provided ID.
